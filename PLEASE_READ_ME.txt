* This website was developed and run with XAMPP server and is assumed will be run locally with XAMPP on other machines.

* Default page on running website is index.html (employee login)
To access Manager and HR login pages, please enter 
"localhost/manager_index.html" and "localhost/hr_index.html" in the browser search bar whilst the application is running.
Please ensure to logout of each dashboard before logging into another one to kill any sessions running previously.

* The database has been exported to an SQL file in this directtory, kindly import this to run app.
* Please find login credentials for HR and Manager in '''login.txt'''. Employee login details can be copied from the database.
